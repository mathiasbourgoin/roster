# Stage S1 — Task tracker: create & fetch

Build an HTTP/JSON service for tracking tasks. Any language/structure (standard library only; Node
v25 and python3 available). Honor the run contract (`start.sh`, foreground, `$PORT`, prints
`LISTENING <port>`). Build it so later features are easy to add.

- `POST /tasks` body `{ "title": <non-empty string> }`
  - 201 `{ id, title, status: "open" }` (id any unique string)
  - 400 if `title` is missing or empty

- `GET /tasks/:id`
  - 200 `{ id, title, status }`
  - 404 if the task does not exist

Include tests.
