# Iteration 2026-05-22-30: Focused Edge Affordances

## Brief

Loop 29 made declared ACL edges focusable. Loop 30 adds a typed affordance
layer for a selected edge so the future MIAOU page can show source and target
pane metadata and bind socket-safe read/write intent descriptions without
inventing new socket mutations yet.

## Acceptance Criteria

- Add a pure focused-edge affordance module.
- Represent socket-safe action intents with typed read/write phantom states.
- Derive edge source and visible target metadata from the actor-visible
  `Dashboard_model.t`.
- Do not expose hidden edge targets in affordance metadata or action lines.
- Render focused-edge source, target, and action lines in the preview panel.
- Expose the selected-edge affordance through `Dashboard_interaction`.
- Keep existing dashboard selection, topology render, and socket protocol APIs
  compatible.
- Cover pure affordance rendering, dashboard model derivation, interaction
  accessor behavior, CLI key replay, and live tmux/socket behavior.
- Run review, QA, live tmux smoke, commit, and push the loop branch.

## Result

Completed.

- Added `Dashboard_edge_affordance` with typed read/write socket intents,
  endpoints, targets, existential actions, and deterministic preview rendering.
- Added `Dashboard_model.edge_affordance`, deriving source/target metadata and
  actions from the already-redacted dashboard model.
- Focused pipeline edges now prepend `Focused edge`, `Edge source`,
  `Edge target`, and `Action` lines to the preview panel.
- Added `Dashboard_interaction.focused_edge_affordance` for future MIAOU key
  bindings.
- Write affordances are actor-bound: future message actions render only when
  the supplied actor is the edge source.
- Focused-edge affordances are hidden outside Pipeline focus, and read intents
  carry the requested preview line count.
- Added tests for pure affordance rendering, redaction of hidden targets,
  interaction access, and CLI `--key p --key Right` output.
- Verified the socket-backed render path with a disposable live tmux workspace.
