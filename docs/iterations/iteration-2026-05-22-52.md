# Iteration 2026-05-22-52: Footer Action Priority

## Brief

Loop 52 keeps the MIAOU launcher footer Herdr-simple by protecting the primary
Enter action from truncation.

## Product Constraint

The normal path is still workspace, agent, Enter. Roster-native authority is
useful only if it does not obscure the key action that starts or refreshes the
selected session.

## Acceptance Criteria

- Detached agents keep a complete `Enter Start` hint in the pinned footer.
- Attached agents keep a complete `Enter Refresh` hint in the pinned footer.
- Capability authority remains visible at normal terminal widths.
- Lower-priority profile and command details drop before the action truncates.
- Runtime, socket, state, and launch behavior remain unchanged.
