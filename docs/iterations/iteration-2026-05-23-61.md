# Iteration 2026-05-23-61: Gated Manage Panel

## Brief

Loop 61 turns the privileged `Manage` hint into a real local TUI surface without
changing the normal start/refresh flow.

## Product Constraint

TA should be as simple as Herdr for ordinary operation, while exposing better
roster/workspace powers only where they apply. Create/connect work starts as a
read-only proposal panel before any persistence or generated-agent mutation is
introduced.

## Acceptance Criteria

- Pressing `m` on a detached privileged agent opens a `Manage` panel.
- The panel shows the actor, authority, create-agent proposal, connect-agents
  proposal, and current access.
- Pressing `m` on an ordinary agent does not leave the launch panel.
- Selection changes or starting the selected agent clear the manage panel;
  refresh preserves it while the same detached privileged agent remains
  selected.
- `Enter Start` and `Enter Refresh` remain the primary launch actions.
