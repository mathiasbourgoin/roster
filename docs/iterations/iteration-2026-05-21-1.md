# Iteration 2026-05-21-1: Bootstrap TA

## Brief

Create the initial TA project inside `agent-roster` without changing the root
TypeScript build. The slice must compile independently and establish the typed
core needed for future MIAOU, socket, and MCP work.

## Acceptance Criteria

- OCaml subproject exists under `ocaml/agent-manager/`.
- Config parser validates workspace, view, agent, link, and permission data.
- CLI can validate a TA config and run a tmux smoke diagnostic.
- Unit tests cover valid config, invalid links, duplicate agents, ID rejection,
  and tmux command construction.
- QA records tmux execution evidence before commit.
