# Iteration 2026-05-23-56: Bounded Picker Profiles

## Brief

Loop 56 caps launch profile labels in the MIAOU agent picker so custom
executables cannot distort the first-screen table.

## Product Constraint

The picker should help the operator choose an agent quickly. Full command
details belong in the launch panel and footer; the table needs short, stable
labels.

## Acceptance Criteria

- Built-in profile labels such as `Codex` and `shell` remain unchanged.
- Long custom profile labels are capped in the agent picker.
- The picker remains an `Agent` plus `Profile` table, without reintroducing a
  status column.
- The selected launch panel and footer still reflect the actual configured
  command.
- Runtime, socket, state, and config behavior remain unchanged.

## Result

Completed.

- Added a bounded picker profile label helper.
- Added headless coverage for a long custom executable, confirming the picker
  shows `very-long...` while hiding the uncapped custom name.
- Kept the full selected launch command in the selected panel/footer behavior.
