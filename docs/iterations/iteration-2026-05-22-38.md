## Review Notes

The main design choice was to commit `.harness/` for this repository. Runtime
surfaces remain generated, but committing the canonical harness avoids a false
source-of-truth where `opencode.json` changes without the manifest that
produced it.

Independent review initially blocked on fresh install breakage, false OpenCode
compatibility, hardcoded broad OpenCode permissions, and a missing
`human-validation` row in `AGENTS.md`. Those findings were fixed before commit.

Generated `index.json` changed during `npm test` and is intentionally left out
of the commit set for this loop.
