# Iteration 2026-05-23-53: Selection-First Launch Panel

## Brief

Loop 53 moves the MIAOU first screen closer to a Herdr-simple launcher. Detached
agents now show a compact launch summary instead of the full diagnostics panel.

## Product Constraint

TA should feel like two selections and Enter. Workspace, roster identity,
profile, and authority should clarify the launch choice without making the
default screen feel like a config inspector.

## Acceptance Criteria

- Detached selected agents render a `Launch` panel with workspace, agent,
  roster agent, launch profile, authority, privileges, capabilities, and
  connections.
- Left/Right and Up/Down update both the launch panel and pinned footer.
- The pinned footer remains the primary action source and keeps complete
  `Enter Start`.
- Live or attached agents keep the preview-first `Agent detail` behavior and
  complete `Enter Refresh`.
- Pipeline focus remains unchanged.
- No runtime, socket, state, or config mutation behavior changes.

## Result

Completed.

- Added a detached-agent launch summary in the MIAOU view.
- Kept live agents on preview plus `Agent detail`, preserving pane diagnostics
  only after a session exists.
- Added headless coverage for launch panel visibility, Right/Down selection
  movement, tiny-height footer pinning, harness-derived privileged agents, QA
  agents, and attached refresh behavior.
