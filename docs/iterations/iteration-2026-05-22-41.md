# Iteration 2026-05-22-41: App-First Startup

## Brief

Loop 41 keeps TA moving toward the Herdr-simple bar: the first path should be
open the app, select an agent, press the primary action. The powerful `tactl`
and socket flows remain available, but they should not be the first thing the
user has to understand.

## Acceptance Criteria

- Make `ta --help`, `tactl --help`, and `tactl quickstart` teach the normal
  TUI path before lower-level state and launch commands.
- Make the startup guide explain that config-backed startup auto-creates
  `.ta-state.json`.
- Keep advanced `tactl state save`, `launch start`, and socket examples
  discoverable as fallbacks.
- Add a compact MIAOU selected-agent action line so the visible TUI says what
  the primary key does.
- Cover the action line in normal, collapsed, and post-start MIAOU headless
  frames.

## Result

Completed.

- The quickstart path now starts with copying or discovering a config, running
  `dune exec ta`, selecting an agent, and pressing `s`.
- Manual state-save and launch-start commands moved behind an advanced CLI
  fallback section.
- `ta` help now includes a normal TUI flow and documents that config-only
  startup opens a state-backed dashboard.
- The MIAOU dashboard now renders `s Start <agent>` for detached agents and
  `r Refresh | attached <pane>` for attached agents in the sidebar and detail
  pane.
- Headless MIAOU tests assert the start action, the collapsed-width action, and
  the attached post-start action.

## Validation

- `opam exec -- dune build @all @runtest`
- MIAOU headless dashboard frame shows `s Start lead`.
- MIAOU headless 39-column collapsed frame still shows `s Start lead`.
- MIAOU headless direct-start frame shows `Runtime LIVE`, `Status running`, and
  `r Refresh | attached`.

## Review Notes

This loop deliberately does not add a new modal or command palette yet. The
next product step is the real two-selection start/create/connect workflow. This
slice clears the first impression so that the TUI, not the CLI harness, is the
default mental model.

Generated `index.json` remains unstaged.
