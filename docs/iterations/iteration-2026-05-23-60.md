# Iteration 2026-05-23-60: Capability-Gated Manage Hint

## Brief

Loop 60 makes TA's roster-native creation and connection powers discoverable
without adding ceremony to ordinary agent launch.

## Product Constraint

TA should stay as simple as Herdr for start/refresh. Extra power belongs only
on agents that can use it, such as `tech-lead` and `recruiter`, and should not
turn every agent selection into a configuration panel.

## Acceptance Criteria

- Agents with `create-agent` and/or `connect-agents` capability show a compact
  `Manage` hint in the detached launch panel.
- Agents without those capabilities do not show the `Manage` row.
- The primary footer remains `Enter Start`/`Enter Refresh`.
- Attached-agent diagnostics still show raw capabilities.
- Existing start, refresh, collapsed layout, and harness startup behavior remain
  unchanged.
